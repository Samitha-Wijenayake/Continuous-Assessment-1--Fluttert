## **Getting Started**

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Samitha-Wijenayake/Continuous-Assessment-1--Fluttert.git
   ```

2. **Navigate to the Directory**:
   ```bash
   cd myJokesApp
   ```

3. **Fetch Dependencies**:
   ```bash
   flutter pub get
   ```

4. **Run the App**:
   ```bash
   flutter run
   ```
